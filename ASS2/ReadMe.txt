Consist of three Files
	data.txt 
		Database file for the script.
		Will be recreated by the script if not found.
	main.py
		Provides the main menu to the program
	database.py
		Handles all interactions with the database(data.txt)
		Prompts user for appropriate data.

Data is rewritten to file every time an employee is either added or removed.
Data is loaded into a list of dictionary at the begining of execution. 
Therefore entries may not be manually added while the program is executing
